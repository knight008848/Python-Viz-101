# 📊 Python-Viz-101

![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)

欢迎来到 **Python-Viz-101**！🚀 这里是 Python 数据可视化从零入门的配套精简代码仓库。

本项目致力于用最少的代码和最清晰的示例，带你快速上手 Python 数据可视化。告别繁琐的理论，直接运行代码，看图说话！🎨

## 🎯 包含内容

* 📈 **基础图表**：柱状图、折线图、饼图等极简代码模板。
* ✨ **图表美化**：标题、图例、数据标签的快速配置方法。
* 💼 **拿来即用**：可以直接复制到你的日常工作或报告中的脚本。

## 🚀 快速开始

1. **克隆仓库到本地：**
```bash
git clone [https://github.com/your-username/Python-Viz-101.git](https://github.com/your-username/Python-Viz-101.git)
cd Python-Viz-101
```

2. **安装基础依赖（建议在虚拟环境中）：**
```bash
pip install matplotlib numpy pandas
```

3. **运行代码：** 打开对应的 `.py` 脚本或 Jupyter Notebook，点击运行即可看到图表！💻
